# init-ubuntu-for-docker
Init system ubuntu for docker

## Install step
* git clone https://github.com/jtleon/init-ubuntu-for-docker.git
* cd init-ubuntu-for-docker/
* sudo chmod a+x init-ubuntu.sh
* ./init-ubuntu.sh
* run-server
